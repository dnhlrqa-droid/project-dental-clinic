# Dental Clinic (React + Tailwind)

Project scaffold for a responsive dental clinic landing site (Arabic RTL) built with Vite + React + Tailwind.

Quick start:

1. Open a terminal in `dental-clinic`.
2. Install dependencies:

```bash
npm install
```

3. Start dev server:

```bash
npm run dev
```

Notes:
- The booking form is a demo (no backend). It shows an alert on submit.
- Styling uses Tailwind; scroll animations use AOS.
